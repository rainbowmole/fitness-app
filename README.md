# Fitness App

A modern web application designed to help users track, manage, and achieve their fitness goals. Built with a focus on usability and visual appeal.

## Features

- 💪 **Workout Tracking** - Log and monitor your workouts
- 📊 **Progress Analytics** - Visualize your fitness journey over time
- 🎯 **Goal Setting** - Create and track personalized fitness goals
- 📱 **Responsive Design** - Works seamlessly across all devices
- 🎨 **Clean Interface** - User-friendly and intuitive design

## Tech Stack

The application is built with:

- **JavaScript (66.3%)** - Core application logic and interactivity
- **CSS (17.5%)** - Styling and responsive layouts
- **HTML (16.2%)** - Markup and page structure

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies required for the web version

### Installation

1. Clone the repository:
```bash
git clone https://github.com/rainbowmole/fitness-app.git
cd fitness-app
```

2. Open the application in your browser:
```bash
# Option 1: Double-click index.html
# Option 2: Use a local server
python -m http.server 8000
# Then visit http://localhost:8000
```

## Usage

1. **Getting Started** - Open the app and create your profile
2. **Log Workouts** - Record your daily exercises and activities
3. **Track Progress** - View your statistics and performance metrics
4. **Set Goals** - Create fitness targets and monitor your achievements
